## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Nestjs

Nestjs는 반드시 하나만 존재해야하는 **`Main`** 을 가지고 있고, `Module`,`Controller`,`Service`로 기본 구성이 되어있습니다.  

### install & Create Project

**[install]**
```shell
$ npm -g install @nestjs/cli
```

**[Create Project]**
```shell
$ nest new [projectName]
```

### Module  

Module은 `@Module()` 데코레이터로 표기하며, Nest가 애플리케이션 구조를 구성하는데 사용하는 메타 데이터를 제공합니다. 모든 nest application은 `하나 이상의 Module`을 갖습니다. 모듈의 종류는 다음과 같습니다.

- Root Module  

- Feature Module  

- Share Module

- Global Module

- Dynamic Module
